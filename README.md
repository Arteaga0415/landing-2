## Installation

Here are the steps you need to follow to install the dependencies.

1.**cd** into the template directory then run this command to install all the dependencies
    
```bash
    npm install
```
    
or
    
```bash
    yarn install
 ```


You can start the project on the local server
    
```bash
    npm run dev
 ```
